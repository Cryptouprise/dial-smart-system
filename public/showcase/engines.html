// UCB1 variant selection const variant = await select_sms_variant( userId, "nurture", campaignId ); // Results after 500 sends: → Variant A: 4.2% reply // underperforming → Variant B: 11.8% reply // WINNER → Variant C: 8.1% reply // decent // AI auto-rewrites Variant A → Variant D: "New AI-generated copy"

UCB1 Bandit Selection

Balances exploration vs exploitation automatically. With < 20 sends, explores randomly. Then shifts to proven winners.

No lead ever falls off. When all playbook rules are exhausted, the perpetual engine takes over with adaptive-gap touches, channel rotation, and value-first messaging. Forever.

// Perpetual follow-up logic const gap = Math.min( minGap + (touchCount * 3), maxGap // caps at 30 days ); // Touch #1: 7 days (SMS) // Touch #2: 10 days (Call) // Touch #3: 13 days (SMS) // Touch #8: 28 days (Call) // Touch #9+: 30 days (alternating) → No lead left behind. Ever.
